# 🛒 Turkmen E-Commerce Tech

Modern Turkmen e-commerce web application for computer technology built with Next.js and Tailwind CSS.

---

## 🚀 Features

- 🧭 Modern Navbar with multi-level dropdown
- 🔍 Product search system
- 🛍️ Shopping cart functionality
- 📦 Product listing & detail pages
- 🌐 Multi-language support (TM / RU / EN)
- 🎨 Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

---
